# ChangingToSwift

###ChangingToSwift01

注意：swift程序不需要进行引用，直接调用就可以了。

###ChangingToSwift02 
创建第二个swift程序，打开摄像头拍照录像，UIImagePickerViewController 和 AVFoundation ，使用storyboard进行布局。

###ChangingToSwift03 
创建第三个swift程序，创建**交互性强**的本地通知，通知的整个流程和相关方法属性的介绍。
